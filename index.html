tombraider.js

A free javascript library for creating 3D menus for websites.
Created by namelivia.

Features

This is a Javascript library for generating 3D spinning menus for your websites.
You can define your own objects, transitions, controls and actions.
It is free and open source (GPLv2).

An example of a 3D menu similar to the original Tomb Raider 2 menu, it contains some weapons and a similar background. The menu control is binded to two html buttons and also the name of the selected element is displayed in a html tag under the menu.

An example of a 3D menu featuring the logos of some interesting projects. The menu control is binded to the keyboard, also the enter key is binded to the action of every item on the menu, which is going to the project's website.

This example features a 3D menu created out of SVG shapes to 3D icons using Blender. It also demonstrates events risen from an object action, that can be captured and custom javascript functions can be called when activating a menu option.
